# 安装与运行程序

- 克隆项目

`> $ git clone https://github.com/A-Jinsc/MyCloudDisk.git`

- 安装依赖

`> $ npm install `

- 在项目目录下启动Vite服务

`> $ npm run dev`

- 发布项目命令

`> $ npm run build`
